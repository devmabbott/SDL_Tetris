Detailed Description

Base mesh interface to store and draw geometry with attributes.

At least the meshArray "vertices" needs to be initialized. All arrays are float. For a 2-dimensional mesh, set "float_per_vertex" to 2. Default is 3. To declare anti-clockwise order, set "winding" to GL_CW. Default is GL_CCW. The following arrays can also be defined: normals, colors, texcords. This is not an abstract class - Can be instantiated to construct meshes from scratch. For a simple example of creating a derived mesh, please refer to cube_mesh.h file.
