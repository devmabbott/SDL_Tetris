Detailed Description

Animation base class

Provides a common interface for animations Any child class must call this base class constructor manually in order to set up the animation properly To implement an animation, override the "animate()" method. This method is called by transform(), and takes a parameter value between [0..1] where 0 is the starting state

Member Enumeration Documentation

enum Animation::RepeatType

Represents the animations repeat type. NONE is a simple animation that runs once, ONE_WAY starts in the beginning phase when finished, TWO_WAY is animated in both direction
